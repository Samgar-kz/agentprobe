FEATURE SUMMARY
================================================================================

1. STRUCTURED LOGGING
   - JSON format support for machine parsing
   - File rotation (configurable size, backup count)
   - Extra fields: attack_id, confidence, latency_ms, tokens, cost, http_status
   - Log levels: DEBUG, INFO, WARNING, ERROR
   - Console + file output

2. METRICS TRACKING
   - Oracle metrics: call count, tokens, latency, cost
   - HTTP metrics: request count, latency, success rate
   - Scan metrics: aggregate stats with throughput calculation
   - Cost calculation for 4 LLM models
   - Confidence tracking and averaging

3. COST TRACKING
   - Model pricing: gpt-4o-mini, gpt-4o, claude-3-haiku, gemini-1.5-flash
   - Token-based cost calculation: (tokens / 1M) × price_per_model
   - Cost formatting: µUSD, mUSD, USD based on amount
   - Example: 12,600 tokens × $0.15/1M = $0.00189

4. CLI ENHANCEMENTS
   - Verbose mode: --verbose (0/1/2) with stackable -v flags
   - Logging: --log-file PATH with automatic rotation
   - Reporting: --json-report PATH, --out-json (legacy)
   - Exit codes: 0=success, 1=error, 2=vulnerabilities, 3=config
   - Fail threshold: --fail-threshold (0-1) for vulnerability detection

5. JSON REPORT FORMAT
   - Scan ID (UUID) and timestamp (ISO8601)
   - Statistics: attacks, hits, cost, throughput, confidence, tokens
   - Results: per-attack success, confidence, evidence
   - Category breakdown: by_category with hit rates
   - Error tracking: separate errors section
   - Metrics: oracle, HTTP, cost data

USAGE EXAMPLES
================================================================================

CLI:
  # Basic with metrics
  agentprobe scan --target dummy -v --json-report report.json
  
  # Debug with logs
  agentprobe scan --target dummy -vv --log-file scan.log --json-logs
  
  # Production scan
  agentprobe scan --target http --endpoint http://api/chat -v \
    --log-file /var/log/scan.log --json-report /var/log/report.json

Python:
  target = DummyVulnerableAgent()
  report, metrics = run_scan(target, track_metrics=True)
  print(f"Cost: {metrics.cost_str}, Speed: {metrics.throughput:.1f} attacks/sec")
  write_json(report, "report.json", metrics=metrics)
